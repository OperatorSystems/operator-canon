ADR-003: Operator is an AI Operating System

Status: Accepted

Context

Traditional productivity software stores information.

Traditional project management software tracks work.

Traditional AI products answer questions.

Operator exists to help people and organizations execute.

Decision

Operator is an AI Operating System that transforms unstructured information into coordinated execution.

Every feature must support one or more of these functions:

* Capture
* Understand
* Plan
* Execute
* Adapt
* Learn

Operator is not a chatbot.

Operator is not a dashboard.

Operator is an execution platform.

Implications

Every page must reduce cognitive load.

AI should recommend rather than merely summarize.

Users approve meaningful changes.

Execution takes priority over information storage.

The system should scale from one person to enterprise organizations without architectural changes.

Principles

Capture reality.

Build understanding.

Generate a campaign.

Coordinate execution.

Continuously adapt.

Preserve organizational knowledge.
