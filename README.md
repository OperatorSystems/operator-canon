# Operator Canon

> The governing doctrine of Operator Systems.

The Operator Canon is the permanent institutional knowledge of Operator.

It defines the principles, doctrine, architecture, engineering philosophy, and long-term vision that guide every product decision.

---

## Purpose

Software changes.

Principles endure.

The Canon exists to ensure every future version of Operator remains aligned with its founding philosophy.
